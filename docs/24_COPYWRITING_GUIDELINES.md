# Harmonix V2

# Copywriting Guidelines

Version 2.0

---

# Purpose

This document defines the writing standards for all text inside Harmonix V2.

Good copy is invisible.

It does not draw attention to itself.

It simply helps users move forward with confidence.

---

# Writing Philosophy

Every word should earn its place.

If removing a word does not change meaning,

remove it.

---

# Voice

Harmonix speaks like a trusted financial advisor.

Not like a startup.

Not like a crypto project.

Not like a robot.

---

Voice qualities:

Calm

Precise

Honest

Helpful

Professional

---

| Context | Tone |
|---------|------|
| Onboarding | Welcoming, simple |
| Portfolio | Confident, informative |
| Deposit | Clear, reassuring |
| Withdraw | Honest, patient |
| Error | Calm, helpful |
| Success | Warm, affirming |
| Empty State | Encouraging |
| Notification | Direct, factual |

# Language Rules

## Rule 1: Business language only

Use financial language.

Not blockchain language.

| Avoid | Use Instead |
|-------|------------|
| Redeem | Withdraw |
| Claim | Receive |
| Operator Fulfill | Processing |
| Settlement | Transfer |
| Mint Shares | Invest |
| Burn Shares | Withdraw |
| Vault Shares | Your Position |
| TX Hash | Transaction |

---

## Rule 2: Active voice

Avoid passive constructions.

Bad

"Your withdrawal has been submitted by the system."

Good

"Your withdrawal is processing."

---

## Rule 3: Plain numbers

Bad

"The TVL of this vault is approximately $12,345,678.23 USD."

Good

"$12.3M total value locked."

---

## Rule 4: Present tense for status

Bad

"Your deposit was completed successfully."

Good

"Your deposit is complete."

---

## Rule 5: Address users directly

Use "you" and "your."

Bad

"User portfolio summary"

Good

"Your portfolio"

---

## Rule 6: No jargon without explanation

If a technical term must appear,

explain it in parentheses or a tooltip.

Bad

"Current epoch APY: 14.2%"

Good

"Current APY: 14.2%"

---

# Number Formatting

Large numbers use shorthand.

| Value | Display |
|-------|---------|
| 1,000 | $1K |
| 1,000,000 | $1M |
| 1,000,000,000 | $1B |

Percentages always show one decimal place.

Example: 12.4% not 12% or 12.42%

Currency amounts show two decimal places for small values.

Example: $0.82 not $0.8

---

# Button Copy

Buttons should be verb + noun.

| Context | Good | Bad |
|---------|------|-----|
| Investing | Deposit Now | Submit |
| Exiting | Withdraw | Confirm |
| Exploring | Explore Vaults | Click Here |
| Confirming | Confirm Deposit | OK |
| Canceling | Cancel | No |

Primary buttons always name the specific action.

Never use generic labels like "Submit" or "Continue" alone.

---

# Error Messages

Every error must answer three questions:

What happened?

Why?

What should the user do next?

---

| Bad | Good |
|-----|------|
| Transaction Failed | Transaction failed — insufficient balance. Check your wallet and try again. |
| Unknown Error | Something went wrong. Please try again or contact support. |
| Invalid Input | Amount exceeds your available balance of 4,200 USDC. |

---

# Success Messages

Success messages should feel complete.

Not just technical.

Bad

"Transaction confirmed."

Good

"5,000 USDC deposited into Delta Neutral USDC. Your position is now active."

---

# Empty State Copy

Empty states teach.

Not disappoint.

| Context | Message |
|---------|---------|
| No positions | You haven't invested in any vaults yet. Explore available opportunities to start earning. |
| No history | Your activity will appear here after your first deposit. |
| No notifications | You're all caught up. We'll notify you when something important happens. |
| No rewards | Join a campaign to start earning rewards on top of your yield. |
| No search results | No vaults match your filters. Try adjusting your search. |

---

# Notification Copy

Notifications are factual.

Not marketing.

| Event | Copy |
|-------|------|
| Deposit completed | 5,000 USDC deposited into Delta Neutral USDC. |
| Withdraw completed | 5,000 USDC transferred to your wallet. |
| Reward available | You have 250 HMX points available to claim. |
| Campaign started | The Summer Yield Campaign is now live. Earn 2× points until July 31. |
| Maintenance | Withdrawals are temporarily paused for maintenance. Expected to resume in 2 hours. |

---

# Loading Copy

Loading states should explain the current operation.

| Context | Copy |
|---------|------|
| Deposit | Submitting your deposit... |
| Withdraw | Processing your withdrawal... |
| Wallet signing | Waiting for wallet confirmation... |
| Data loading | Loading your portfolio... |
| Transaction pending | Transaction submitted. Waiting for confirmation... |

Never show only "Loading..."

---

# Tooltip Copy

Tooltips explain unfamiliar terms.

Rules:

Under 80 characters.

No punctuation at the end.

Plain language.

| Term | Tooltip |
|------|---------|
| APY | Annual Percentage Yield — your estimated yearly return |
| TVL | Total Value Locked — total assets deposited in this vault |
| Risk Level | How likely this strategy is to experience losses |

---

# Page Titles

Page titles are short and goal-oriented.

| Page | Title |
|------|-------|
| Home | Earn institutional-grade yield |
| Earn | Available Vaults |
| Portfolio | Your Portfolio |
| Rewards | Your Rewards |
| Settings | Settings |

---

# Section Titles

Section titles are noun phrases.

Not questions.

| Bad | Good |
|-----|------|
| What are you earning? | Your Earnings |
| Where is your money? | Active Positions |
| How are rewards working? | Reward Summary |

---

# Final Copywriting Principle

Users should never need to re-read a sentence.

If a sentence requires re-reading,

rewrite it.

Clarity is more important than cleverness.

Brevity is more important than completeness.

Confidence is the product.

Copy should reinforce it.
